BerryNoid
======
An HTML5 Arkanoid-type Game
* See a [demo](https://anderrv.github.io/berrynoid/)

Add [Service Worker](https://jakearchibald.github.io/isserviceworkerready/) support for offline. Works on Chrome 41+
